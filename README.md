# OpenBidHub side-proj

공공 입찰 정보 공개 및 통합 구매시스템 연동 플랫폼

## 기술 스택

### Frontend
- React + TypeScript + Vite
- Tailwind CSS
- Axios, React Query, React Router DOM
- React Hot Toast

### Backend
- Spring Boot
- MariaDB
- RESTful API

## 구조

- `main`: 배포 기준
- `front`: 프론트엔드 통합
- `back`: 백엔드 통합
- `feature/*`: 기능별 브랜치

## 목표

- 온비드 API 기반 공매물건 실시간 제공
- 입찰 일정 및 정보 공개
- 통합 구매시스템 연동으로 기관 구매 효율화
